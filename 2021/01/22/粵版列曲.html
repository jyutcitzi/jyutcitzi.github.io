𠄡知大家識𠄡識解答兩題  ？

題一：   []

𠄡該一個叫 nums 旡 入面搵返兩個加埋等於 target 旡數字旡相應。你可以當每個淨係得一個，而亇個係𠄡會重複用同一個 。答案入面旡隨便你。

例一：
: nums = [2,7,11,15], target = 9
: [0,1]
解釋：答案係 [0,1]，因爲 nums[0] + nums[1] == 9

例二：
: nums = [3,2,4], target = 6
: [1,2]

:

  • 2 <= nums.length <= 103
  • -109 <= nums[i] <= 109
  • -109 <= target <= 109
  • 淨係得一個答案係

：https://leetcode.com/problems/two-sum

題二：    []

而家攞兩個𠄡係吉旡  代表兩個 。 入面旡反轉，而每個入面淨係得一個。𠄡該將兩個數字加埋，同埋用返一個    返個。

例一：
2 -> 4 -> 3, 5 -> 6 -> 4 => 7 -> 0 -> 8
: l1 = [2,4,3], l2 = [5,6,4]
: [7,0,8]
解釋: 342 + 465 = 807.

例二：
: l1 = [0], l2 = [0]
: [0]

例三：
: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
: [8,9,9,9,0,0,0,1]

:

  • 每個 入面有一至一百個
  • 0 <= Node.val <= 9
  • 代表數字旡係一定𠄡會有 

：https://leetcode.com/problems/add-two-numbers
